SinGAN, an ablation study.

(2019, Coursework for CS7015: Deeplearning)

This project was completed as a part of course project in Deep Learning. SinGAN (ICCV 2019) proposes an unconditional generative model to make photo realistic similar images from a single natural image. This can be used for many purposes like image impainting, super resolution, random sample generation etc. Apart from doing an ablation study on the algorithm, we also extended the network to generate similar samples of images from edge maps
